1. To deploy Redis instance, apply this template:

---
apiVersion: middleware.alauda.io/v1
kind: Redis
metadata:
  name: redis-failover
spec:
  arch: sentinel
  customConfig:
    save: 60 10000 300 100 600 1
  exporter:
    enabled: true
  passwordSecret: redis-password
  persistent:
    storageClassName: sc-topolvm
  persistentSize: 1Gi
  replicas:
    sentinel:
      master: 1
      slave: 1
  resources:
    limits:
      cpu: 300m
      memory: 300Mi
    requests:
      cpu: 300m
      memory: 300Mi
  sentinel:
    replicas: 3
  version: "6.0"
---

Wait the Redis Sentinel instance to start up for about 3 minutes.
